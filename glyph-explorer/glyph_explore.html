Glyph Grid (edit pixels)

This explorer uses the common SSD1306/ThingPulse font packing: column-major. For a glyph of height H, each column uses bytesPerCol = ceil(H/8). Bytes are laid out as: col0(byte0..byteN), col1(byte0..byteN), ....

Notes:
  • Bits in each byte map to vertical pixels (rows) in that column.
